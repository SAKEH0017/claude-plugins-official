# reduce-ai

降低中文学术论文 AIGC 检测率的 Claude Code 技能插件。

## 功能

把 AI 生成痕迹过重的论文文本改写成更自然的真人写作风格，适用于：

- 本科毕业论文的降重和去 AI 痕迹
- 学术文本润色，通过 AIGC 检测
- 中文论文的自然化改写

## 核心策略

1. **添加口语化虚词** — 的、了、到、过、会、有、能、把 等自然语气词
2. **替换 AI 过渡词** — 把"首先/其次/最后"改成"第一点/二是/还需要提到的是"
3. **打破句式平衡** — 打乱排比和对称结构，让句子节奏不规则
4. **术语密集句缩短，日常句拉长** — 制造自然的人类写作节奏
5. **保持学术严谨** — 绝对不改专业术语、数据、引用和原意

## 安装

```bash
# 从官方市场安装
/plugin install reduce-ai@claude-plugins-official

# 或手动安装：将 skills/reduce-ai/SKILL.md 复制到 ~/.claude/skills/ 下
```

## 使用

在 Claude Code 对话中粘贴需要改写的文本，或指定论文文件路径。插件会自动触发并对文本进行自然化改写。

触发关键词：降低AI、改AIGC、润色论文、降重、去AI痕迹、改写论文
