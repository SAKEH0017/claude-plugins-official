# thesis-polish

A Claude Code skill plugin that improves Chinese academic thesis writing by making AI-assisted drafts sound more natural and human-like.

## Purpose

Naturalize academic text to improve writing quality, suitable for:

- Undergraduate thesis writing refinement
- Academic text polishing for more authentic expression
- Improving the readability and naturalness of Chinese academic papers

## Core Strategies

1. **Add Natural Filler Words** — Incorporate 的, 了, 到, 会, 能, 把 and other natural function words
2. **Replace Mechanical Transitions** — Swap 首先/其次/最后 with more varied alternatives
3. **Vary Sentence Rhythm** — Break uniform parallelism, mix short and long sentences
4. **Adjust Sentence Density** — Shorten term-heavy sentences, expand everyday-language sentences
5. **Preserve Academic Integrity** — Professional terminology, data, citations, and original meaning are never altered

## Installation

```bash
# From the official marketplace (after approval)
/plugin install thesis-polish@claude-plugins-official

# Manual install: copy skills/thesis-polish/SKILL.md to ~/.claude/skills/
```

## Usage

Paste the text you want polished in a Claude Code conversation, or provide a file path. The plugin triggers automatically and rewrites the text for more natural, human-like expression.
